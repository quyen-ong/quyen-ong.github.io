# Portfolio Website

This repository contains my personal portfolio website. It expands on my resume with a clean presentation of my background, experience, education, and projects while also giving me space to practice HTML and CSS.

## About the site

The website is a lightweight static portfolio built with:

- HTML
- CSS
- GitHub Pages

It highlights:

- My technical skills
- Work experience
- Education
- Engineering and software projects

## Project structure

- `index.html` contains the main content and page structure
- `styles.css` contains the layout, colors, typography, and responsive styling

## Running locally

Because this is a static site, you can open `index.html` directly in a browser to preview it locally.

## Purpose

I built this project to:

- Create an online version of my portfolio
- Present my experience in a more visual format than a resume alone
- Improve my front-end development skills through hands-on practice
